# BigCodeLLM-FT-Proj

BigCode LLM Fine-Tuning Project

## Project Overview

This repository contains code for fine-tuning large language models (LLMs) for code generation tasks. The project is structured with multiple branches for collaborative development.

## Repository Structure

- **main**: Production-ready code
- **dev-me**: Development branch for feature implementation
- **dev-friend**: Development branch for additional features

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/nobita-0002/BigCodeLLM-FT-Proj.git
   ```

2. Checkout the desired branch:
   ```bash
   git checkout dev-me  # or dev-friend
   ```

3. Install dependencies (if any):
   ```bash
   pip install -r requirements.txt
   ```

## Contributing

Please create pull requests from development branches to main for code review and integration.
